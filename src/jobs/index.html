Junior Software Developer

Remote position | 25-28k€/year | Multiple perks

We are in need of a junior developer that wants to work and grow with us! We will train you in all the technologies that we use over time and you will be able to work from home! Plus we offer some sweet perks that will make it worthwile.

We need a junior developer that is willing to grow with us over time and make super cool software and experiment with new tech!

Requirements:

  • HTML5 and CSS3 knowledge
  • Knowledge in at least one of these languages: Python, Rust, C
  • Basic knowledge of SQL / NoSQL databases
  • GNU/Linux

Nice to have:

  • Git version control
  • Docker
  • Terraform or Ansible
  • Kownledge of one of these frameworks: Django 4.x / FastAPI / Flask / Falcon

What do we offer?

  • Salary: 25k€ base salary, up to 28k€ based on objectives
  • Holidays: 40 days of holidays
  • 100% Remote. Work from wherever you want :)
  • Over time other perks will accumulate, like work time reduction to 6h/day, sweet, right?

As an equal opportunity employer and firm believers that everyone deserves a chance, feel free to apply even if you don't meet all the criteria. We can't promise we will hire you but you will definitely not be overlooked!
